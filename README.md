# Workshop Interview Prep
This is a minimal tool designed to help users prepare for interviews.

## Features
- Practice questions for various topics.
- Timed quizzes to simulate real interview conditions.

## Tech Stack
- JavaScript
- HTML
- CSS

## Usage
Clone the repo and open `index.html` in your browser to start practicing.

## Future Plans
- Add more question categories.
- Include a scoring system to track progress.
